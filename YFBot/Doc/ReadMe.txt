Идеально для использования с "Кавказом", можно в комплекте с дронами/турелями.
Игра в оконном, окно не сворачивать, всё работает пока окно игры активно.
При переключении на другое окно - утилита приостанавлявает работу, при возвращении в окно игры - возобновляет.

Weapon period - величина паузы в секундах между запуском дронов/турели.
Надо настроить дронов/турели на клавишу "1",

Есть три режима работы.
	сruiseLogic - для стратегии "только вперед!",
	defendLogic - для режима АФК-фарма с "Кавказом",
	customScript - для использования пользовательских скриптов.

COB Script:

	Forward; - вперед,
	Back; - назад,
	Left; - налево,
	Right; - направо,
	Stop; - стоп (сброс всех осей),
	Delay 300; - задержка между действиями (1000 = 1 секунда),
	RandomDelay 3000; - задержка между действиями, от 0 до заданного значения (3000 = 3 секунды в данном случае),
	StepUp 3; - перейти на 3 шага назад в скрипте,
	StepDown 3;- перейти на 3 шага вперед в скрипте.

*StepUp и StepDown на данный момент лучше не использовать.

	Пример использования пользовательского скрипта:

		Forward;			// Вперед!
		Delay 1000;			// едем вперед 1 секунду
		Left;				// продолжая ехать вперед, поворачиваем налево
		Delay 500;			// и так 0,5 секунды
		Stop;				// перестаем ускоряться и поворачивать
		Forward;			// возобновляем ускорение
		Right;				// + право руля
		Delay 500;			// и так ещё 0,5 секунды
		Stop;				// перестаем ускоряться и поворачивать
		Back;				// задний ход
		RandomDelay 3000;	// в диапазоне от 0 до 3 секунд едем назад
							// (и всё сначала)
