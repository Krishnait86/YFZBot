Больше никаких отключений за бездействие в Crossout!
Идеально для использования с "Кавказом", можно в комплекте с дронами/турелями.
Работает пока окно игры активно. Необходим оконный режим.
При переключении на другое окно - утилита приостанавлявает работу, при возвращении в окно игры - возобновляет.

Weapon period - величина паузы в секундах между запуском дронов/турелей.
В игре запуск дронов/турелей необходимо назначить на клавишу "1".
При отсутствии дронов/турелей поле не требует изменений (00).
*Не желательно использовать числа кратные 10 или меньше "05".

Есть два преднастроенных паттерна и возможность создать свой:

	сruiseLogic		- режим не для АФК "только вперед!",
	defendLogic		- преднастроенный режим АФК-фарма с "Кавказом",
	customScript	- для использования пользовательских скриптов.

Для пользовательских скриптов используется текстовый файл.
Удобнее расположить его в одной папке с утилитой.
Загружается из меню "File -> Load Script..." (и далее - сам файл)
После чего необходимо выбрать "customScript" и нажать кнопку Start.

COB Script:

	Forward;			- вперед,
	Back;				- назад,
	Left;				- налево,
	Right;				- направо,
	Stop;				- стоп (сброс всех осей),
	Delay 1000;			- фиксированная продолжительность указанного действия в милисекундах,
	RandomDelay 3000;	- продолжительность определяется случайно в интервале от 0 до значения заданного в милисекундах,
	StepUp 3;			- перейти на 3 шага назад в скрипте,
	StepDown 3;			- перейти на 3 шага вперед в скрипте.

*StepUp и StepDown на данный момент лучше не использовать.

	Пример пользовательского скрипта:

		Forward;
		Delay 5000;			// едем вперед 5 секунд
		Left;
		Delay 500;			// добавляем левый поворот и через 0,5 секунды отпускаем его
		Right;
		Delay 500;			// + право руля на 0,5 секунды
		Stop;				// перестаем ускоряться и поворачивать
		Back;
		RandomDelay 3000;	// задний ход на время от 0 до 3 секунд
							// (и весь цикл повторяется)
